# Scraper
Parser Combinator Library in C

## build
```sh
$ git clone https://github.com/rot-E/Scraper.git
$ cd Scraper
$ mkdir build && cd build
$ cmake ..
$ make -j
```

## install
```sh
$ cmake .. -DCMAKE_BUILD_TYPE=Release -DCMAKE_INSTALL_PREFIX=$HOME/.local
$ make install -j
```

## build examples
```sh
$ make examples -j
$ examples/DeclVar
```
